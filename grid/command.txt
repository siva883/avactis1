First Create Event Bus
java -jar selenium-server-4.8.0.jar  event-bus

Session Map
java -jar selenium-server-4.8.0.jar sessions

New Session Queue
java -jar selenium-server-4.8.0.jar sessionqueue

Distributor
java -jar selenium-server-4.8.0.jar distributor --sessions http://localhost:5556 --sessionqueue http://localhost:5559 --bind-bus false

Router
java -jar selenium-server-4.8.0.jar router --sessions http://localhost:5556 --distributor http://localhost:5553 --sessionqueue http://localhost:5559

Create Nodes
java -jar selenium-server-4.8.0.jar node --max-sessions 4 --log-level "fine" --port 7777 --driver-implementation "firefox" --driver-implementation "edge"
